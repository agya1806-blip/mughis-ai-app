# Image Service

**Provider:** Pollinations.ai (default, gratis) / Hugging Face Inference API

**Free Tier:**
- Pollinations.ai: Unlimited, no API key needed
- Hugging Face: FLUX.1-dev, FLUX.1-schnell, SDXL via Inference API

**Models Available:**
- Pollinations.ai (default, fastest)
- black-forest-labs/FLUX.1-dev (kualitas terbaik)
- black-forest-labs/FLUX.1-schnell (cepat)
- stabilityai/stable-diffusion-xl-base-1.0 (SDXL)
- runwayml/stable-diffusion-v1-5 (ringan)

**Resolutions:** 512×512, 768×768, 768×1024, 1024×768, 1024×1024, 1280×720
