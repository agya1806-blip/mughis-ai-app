# Video Service

**Provider:** Hugging Face Inference API

**Free Tier:** Hugging Face Inference API (rate limited, cold start ~20-60s)

**Models Available:**
- stabilityai/stable-video-diffusion-img2vid (default, image-to-video)
- damo-vilab/text-to-video-ms-1.7b (text-to-video)

**Note:** Video generation membutuhkan Hugging Face token. Pollinations tidak mendukung video.
