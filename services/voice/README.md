# Voice Service

**Provider:** Browser Speech Synthesis (default) / Hugging Face Inference API

**Free Tier:**
- Browser SpeechSynthesis: Gratis, offline, tanpa API key
- Hugging Face TTS: Gratis (rate limited)

**Features:**
- Text-to-Speech (TTS): Browser native + Hugging Face (facebook/mms-tts-eng)
- Speech-to-Text (STT): Hugging Face (openai/whisper-small)

**Note:** Browser TTS bekerja offline tanpa API key. STT membutuhkan Hugging Face token.
