# Chat Service

**Provider:** Gemini API (Google AI Studio) / Hugging Face Inference API

**Free Tier:**
- Gemini: 60 requests/minute free (rate limit: 1,500 requests/day)
- Hugging Face: Inference API free tier (rate limited)

**Models Available:**
- `gemini-2.0-flash` (default, via Gemini API)
- `microsoft/DialoGPT-medium` (via Hugging Face)
- `microsoft/DialoGPT-large` (via Hugging Face)

**API Key:** Set via Settings → masukkan Gemini API Key (AIzaSy...) atau Hugging Face token (hf_...)
